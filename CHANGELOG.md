# Changelog

Todos los cambios notables en este proyecto serán documentados aquí.

## [1.0.0] - 2026-05-10

### Añadido
- Detección automática de roles (Usuario vs IA) para ChatGPT, Claude, Gemini, Copilot, DeepSeek
- 5 plantillas inteligentes: Completa, Técnica, Creativa, Resumen, Tutorial
- Editor visual por bloques con selección individual
- Drag & Drop para reordenar bloques
- Edición inline de cualquier bloque
- Búsqueda en tiempo real sobre bloques detectados
- Toggle tema oscuro/claro persistente
- Portapapeles integrado (pegar con un botón)
- Vista previa de Markdown en tiempo real
- Exportación nativa a archivo `.md`
- Atajos de teclado: Ctrl+Enter, Ctrl+S, Ctrl+T, Ctrl+Shift+V
- Build portable para Windows via Electron Builder
- GitHub Actions para releases automáticos multiplataforma
