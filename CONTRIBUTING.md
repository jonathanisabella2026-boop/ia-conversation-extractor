# 🤝 Guía de Contribución

¡Gracias por tu interés en contribuir a **IA Extractor**!

## Cómo contribuir

1. **Fork** el repositorio
2. Crea una rama: `git checkout -b feature/nombre-de-tu-feature`
3. Haz tus cambios y **commit**: `git commit -m "feat: añade nueva función"`
4. **Push** a tu fork: `git push origin feature/nombre-de-tu-feature`
5. Abre un **Pull Request**

## Estándares de código

- Usa nombres descriptivos en inglés para variables y funciones
- Comenta la lógica compleja
- Mantén la compatibilidad con Electron 30+
- Prueba en Windows antes de enviar PRs relacionados con paths

## Reportar bugs

Usa las plantillas de issues en `.github/ISSUE_TEMPLATE/`.
