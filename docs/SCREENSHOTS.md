# 📸 Guía de Screenshots

Para completar el README con capturas de pantalla:

1. Ejecuta la app: `npm start`
2. Pega una conversación de ejemplo
3. Toma screenshots de:
   - **Modo oscuro** (tema por defecto)
   - **Modo claro** (pulsa 🌓)
   - **Editor de bloques** con varios tipos seleccionados
4. Guarda en `assets/screenshots/` como PNG
5. Actualiza los enlaces en `README.md`

### Ejemplo de conversación para demo

```
You: Hola, necesito ayuda para crear una API en Node.js

ChatGPT: ¡Claro! Aquí tienes los pasos:

1. Inicializa el proyecto
   npm init -y

2. Instala Express
   npm install express

3. Crea el servidor
   ```javascript
   const express = require('express');
   const app = express();
   app.listen(3000);
   ```

En resumen: necesitas Node, Express y un archivo server.js.
```
